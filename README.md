# 🌐 Personal Portfolio

A modern, responsive, and interactive personal portfolio website built to showcase my projects, technical skills, achievements, certifications, education, and professional experience.

The portfolio follows a clean UI with smooth animations, responsive layouts, dynamic project loading using JSON, and an integrated contact form powered by EmailJS.

---
## 🚀 Live Demo

🔗 **Portfolio:** 
https://bhagyashreemistari.vercel.app/

---
# ✨ Features

- 🎨 Modern Responsive UI
- 📱 Mobile Friendly Layout
- ⚡ Smooth Animations (AOS)
- 📂 Dynamic Project Loading using JSON
- 🏆 Achievements Section
- 📜 Certifications Section
- 📧 Contact Form using EmailJS
- 🔗 Social Media Integration
- ⬆️ Back to Top Button
- 🚀 Clean Folder Structure
